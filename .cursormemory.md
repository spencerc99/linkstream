# LinkStream Project Memory

## Project Overview

LinkStream is a React application that displays real-time links from Bluesky social media platform using JetStream. It shows embedded content from various domains and tracks domain statistics.

## Recent Changes

### Domain Histogram Component & Country Flags (Latest)

- Created separate `DomainHistogram` component in `src/components/DomainHistogram.tsx`
- Added favicon display using Google's favicon service
- Implemented country flag detection based on TLD (top-level domains)
- Uses country-specific TLDs (.de, .fr, .jp, etc.) and second-level domains (.co.uk, .com.au, etc.)
- Removed specific website patterns in favor of generic TLD-based detection
- Enhanced design with cards, better spacing, scrollable container, and hover effects

### Domain Normalization

- Added `normalizeHostname()` function to remove "www." prefix from hostnames
- Updated domain counting logic to use normalized hostnames
- This ensures that "www.example.com" and "example.com" are treated as the same domain
- Updated `transformUrl()` function to handle normalized domains for URL transformers
- Removed duplicate entries in URL_TRANSFORMERS (e.g., "www.youtube.com")

## Key Components

### StreamView.tsx

- Main component that handles real-time link streaming
- Uses `useJetStream` hook for Bluesky connection
- Displays links in a grid layout with domain statistics sidebar
- Supports pausing/resuming the stream

### URL Transformers

- YouTube: Converts to embed URLs with autoplay
- Twitter/X: Converts to embedded tweet format
- Spotify: Converts to embed format for tracks/albums/playlists
- Now handles both www and non-www versions automatically

### Domain Statistics

- Shows histogram of domains by frequency
- Uses normalized hostnames to group similar domains
- Sorts by count in descending order

## Technical Details

- Built with React + TypeScript + Vite
- Uses Bun as package manager
- Real-time streaming via JetStream API
- Responsive grid layout for link display
- Automatic URL transformation for supported platforms
